## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## new a nest project

1. nest new project-name
2. edit .prettierrc, and add a line : "endOfLine":"auto"
3. npm i @nestjs/swagger
4. add swagger.ts file
5. edit main.ts, and add a line : "setupSwagger(app);"
6. npm run start:dev, and start your nestJS travel.
